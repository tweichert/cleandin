# cleanDIN - Reinigungsfirma Weichert Website

## Beschreibung
Dies ist die offizielle Website für die Reinigungsfirma Weichert, die unter der Marke **cleanDIN** operiert.

## Funktionen
- **Responsive Design**: Die Website passt sich automatisch an verschiedene Bildschirmgrößen an
- **Funktionierendes Kontaktformular**: JavaScript-basiertes Formular mit Validierung
- **Professionelles Design**: Moderne Farbgestaltung mit Blau-Tönen und Gradienten
- **Nicht geschützte Bilder**: Alle Bilder sind frei zugänglich im `images/` Ordner
- **Rechtlich korrektes Impressum**: Vollständiges Impressum gemäß § 5 TMG

## Struktur
```
website/
├── index.html          # Hauptseite
├── style.css           # Stylesheet
├── script.js           # JavaScript für Kontaktformular
└── images/             # Bildordner
    ├── hero.jpg        # Hero-Bild
    ├── office_cleaning.jpeg
    ├── house_cleaning.jpeg
    └── window_cleaning.jpeg
```

## Verwendung
1. Öffnen Sie `index.html` in einem Webbrowser
2. Die Website kann auf jedem Webserver gehostet werden
3. Passen Sie die Platzhalter im Impressum an (Name, Adresse, etc.)

## Anpassungen
- **Impressum**: Ersetzen Sie alle Platzhalter in eckigen Klammern [...] mit Ihren tatsächlichen Daten
- **Kontaktformular**: Aktuell nur Frontend-Validierung. Für echten E-Mail-Versand benötigen Sie ein Backend
- **Bilder**: Alle Bilder können frei ausgetauscht werden

## Rechtliche Hinweise
Die Website enthält den offiziellen Gewerbename "Reinigungsfirma Weichert" im Impressum, mit dem Fantasienamen "cleanDIN" als Marke, wie rechtlich erforderlich.

## Deployment
Die Website kann direkt auf jedem Webserver oder Hosting-Service hochgeladen werden. Keine speziellen Anforderungen außer HTML/CSS/JavaScript-Unterstützung.
